# learn-terraform-hcp-vault

Example Terraform configuration to work with [Deploy HCP Vault Dedicated with
Terraform](https://developer.hashicorp.com/vault/tutorials/manage-hcp-vault-dedicated/deploy-hcp-vault-terraform)
tutorial from HashiCorp.

## Prerequisites 

- HCP account
- HCP service principal
- AWS account

This example configuration is provided as is. Using this configuration may
result in additional charges to your AWS and/or HCP account.
